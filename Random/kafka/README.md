# Steps

1. Create 3 folders in the same directory
 
```
mkdir es-data  kafka-data  zk-data
````

2. Run the docker-compose 
```
docker-compose up -d
```

3. Create a topic named message-log

4. Run the program

```
go run main.go
```

## Prerequisite

- Install docker compose
- Install go lang
- Install Kafka Tool
